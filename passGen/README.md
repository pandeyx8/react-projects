# Random Password Generator

A simple password generator built with React and Vite.

## Features

- Generate random passwords
- Adjust password length with a range slider (6 to 18)
- Include numbers
- Include special characters
- Copy generated password to clipboard

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- Tailwind CSS

## Project Goal

This project was created for practice with React hooks, component state, and basic UI interactions.
