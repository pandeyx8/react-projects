# practiced custom hooks and context API to create a currency changer app. It allows users to select a currency from a dropdown menu and displays the corresponding exchange rate. 

## Features
- Currency selection: Users can choose from a list of currencies to see the exchange rate.
- Real-time exchange rates: The app fetches the latest exchange rates from an API to provide accurate information.
- User-friendly interface: The app has a simple and intuitive design for easy navigation.

## Technologies Used
- React: A JavaScript library for building user interfaces.
- custom hooks: Reusable functions that allow you to use state and other React features without writing a class.
